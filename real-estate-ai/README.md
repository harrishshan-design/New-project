# Real Estate AI

Scaffold for a real estate product with buyer intelligence, agent workflow tools, and AI-assisted services.

## Structure

- `docs/`: product and architecture context
- `features/`: feature briefs
- `backend/`: API, services, routes, models, and utilities
- `frontend/`: UI pages and components

## Next Steps

1. Install backend dependencies with `npm install` inside `backend/`
2. Start the API with `npm run dev`
3. Connect the existing dashboard flows to this API incrementally
